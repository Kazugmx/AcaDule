# 🎓 AcaDule Academic X Schedule

> 学生のためのスケジュール・タスク管理ツール
> Manage your tasks, lessons, and study logs - anywhere, on any platform

---

## Overview

AcaDuleは、**学生の学習生活をサポートするAIOツール**です。
課題・講義スケジュール・学習ログを一元管理し、WebとCLI双方で操作可能。
